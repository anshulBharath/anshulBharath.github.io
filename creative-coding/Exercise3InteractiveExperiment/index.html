Interactive Experiment

Canvas Controls

Keyboard Controls
  • Space: Pause music and movement
  • Enter: Toggle mic input
  • A: Add a square element
  • S: Remove last square element
  • D: Remove all elements
  • F: Toggle fullscreen mode
  • B: Clear background w/o removing elements
  • Bkspce: Remove all elements and clear background
  • P: Take a screenshot of the current Canvas
